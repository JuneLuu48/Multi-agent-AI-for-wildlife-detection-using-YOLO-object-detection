# Multi-agent-AI-for-wildlife-detection-using-YOLO-object-detection

## Problem context
Wildlife-vehicle collisions (WVCs) are increasing in forest-adjacent road and railway areas
These incidents threaten biodiversity, human safety, and cause significant economic losses
Particularly relevant in countries such as Australia and the United States

## Project aim
Develop an AI-powered multi-agent system for real-time wildlife detection
Provide proactive driver alerts to reduce wildlife-vehicle collisions

## System architecture
Integrates a pretrained YOLOv8x deep learning model for animal detection
Uses a coordinated multi-agent framework consisting of:
Perception Agent – detects animals in real time
Threat Assessment Agent – evaluates collision risk
Communication Agent – issues real-time driver alerts
Monitoring Agent – logs events and monitors system performance

## Methodology and evaluation
Model performance evaluated using: Mean Average Precision (mAP); Precision; Recall
Benchmarked against: Other YOLO model variants; Faster R-CNN model

## Key outcomes
Achieves high detection accuracy
Supports real-time processing and scalable deployment
Enables timely interventions to reduce wildlife-vehicle collisions

## Societal and sustainability impact
Contributes to UN Sustainable Development Goal 11 (Sustainable Cities and Communities)
Enhances transport safety in urban-fringe areas
Supports biodiversity conservation
